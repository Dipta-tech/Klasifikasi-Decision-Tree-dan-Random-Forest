Repository ini berisi implementasi algoritma Machine Learning untuk tugas klasifikasi menggunakan metode Decision Tree dan Random Forest.

Dan ini hasil project google collab yang saya buat https://colab.research.google.com/drive/1CTJoL_m0axPDUPMxqUhz_LphKY9dyeJ6?usp=sharing

Proyek ini mencakup seluruh alur analisis data, mulai dari:

Data preprocessing
Eksplorasi data (EDA)
Training model
Evaluasi performa model

Tujuan utama dari proyek ini adalah untuk membandingkan performa kedua algoritma dalam menyelesaikan masalah klasifikasi.

🧠 Algoritma yang Digunakan
1. Decision Tree
Model berbasis pohon keputusan
Mudah dipahami dan divisualisasikan
Rentan terhadap overfitting jika tidak dikontrol
2. Random Forest
Metode ensemble dari banyak Decision Tree
Lebih stabil dan akurat
Mengurangi overfitting

Python

pandas

matplotlib / seaborn
scikit-learn

Perbandingan Model
